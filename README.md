# 🧠 AI Learning Coach

> AI-powered study assistant that reads your syllabus and generates personalized quizzes instantly!

## ✨ Features
- 📄 PDF, TXT, PPTX file support
- 🔍 Auto topic extraction using NLP
- ❓ AI-generated quizzes (5 to 50 questions)
- 📊 Score tracking & performance analysis
- 💬 Built-in AI doubt solver chatbot
- 🎨 Modern dark theme UI

## 🛠️ Tech Stack
- Python
- Streamlit
- Groq API (LLaMA 3.3)
- NLTK
- PyMuPDF
- Python-PPTX

## 🚀 How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Create .env file and add your Groq API key:
   GROQ_API_KEY=your_key_here
4. Run the app:
   streamlit run app.py
